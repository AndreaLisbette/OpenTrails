This tool is best used with the following "input schema"

This tool converts shapefiles that contain data in certain schema.

These schemas are based on data received from park agencies to date. As this tool is used and developed by the community, it will grow to accept a wider variety of data.

To ensure successful use of this converter, we advise you to create shapefiles that contain the following field names and values.

Trail Segments shapefile

From this shapefile, the following files will be created:

  • trail_segments.geojson
  • named_trails.csv
Field Name Values Required?

ID

Alternatives: 'id', 'trailid', 'objectid', 'trail id', 'object id'

String

No

Explanation: This is required in OpenTrails, but not in your upload data. If you don't include one, the Converter will automatically create one for each segment.

Trail

Alternatives: 'trail', 'trailname', 'trail_name'

String

Yes

Explanation: Each segment should be associated with one more more Named Trail. Many segments can (and probably should) be associated with the same trail.
