Upload a .zip file

This validator will evaluate the data you upload for conformance with the OpenTrails specification. To that end, it expects a single .zip file containing:

  • trail_segments.geojson
  • named_trails.csv
  • trailheads.geojson
  • stewards.csv
  • areas.geojson (optional)

If you are missing one or more of these files, you may still upload a .zip to evaluate the data you have included.
