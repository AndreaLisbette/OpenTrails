Your trail segments do not contain a reference to a trail name.

In the current version of this converter tool, we require that the trail segments contains in your shapefile include a trail name. This converter tool extracts those trail names, and creates the named_trails.csv file. Named Trails represent groups of segments, which constitute complete trails. Segments can belong to more than one Named Trail.

For more information, please review the Open Trail System Specification (OpenTrails) in detail.
