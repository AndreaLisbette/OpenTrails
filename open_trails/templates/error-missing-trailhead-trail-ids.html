Trailheads are required to be associated with one or more trail. By definition, they provide visitors access to trails.

OpenTrails is designed to communicate to visitors not just where trails traverse, but also how they can be accessed. This is accomplished primarily through the inclusion of trailheads. By definition, a trailhead provides access to one or more trial.

For more information, please review the Open Trail System Specification (OpenTrails) in detail.

Excerpt from the Open Trail System Specification
Field Name Data Type Required? Description
trail_ids string-encoded array no An array of trail ids, each of which the trailhead provides access to.
