Upload your trail data

Prepare a single ZIP file containing the data you would like to convert, and upload it here.

Upload a zipped shapefile that describes your trail network. This data should be a 'polyline' file that represents the actual paths a trail takes. Most often, many individual features or "segments" are used to described the entirety of a "named trail".

    As you probably know, a "shapefile" is actually a set of several files. To use this converter, we require the following files be included in a single zipped upload:

  • .shp
  • .shx
  • .dbf
  • .prj
