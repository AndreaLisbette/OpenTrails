This tool is under active development. At the moment, this converter converts only trail segments (polylines). Functionality performing conversion of trailheads (points) and areas (polygons) is coming soon.

Understand

This tool will help you convert Shapefiles to OpenTrails, a GeoJSON open data format ideal for web and mobile applications.

OpenTrails consists of the following files:

  • trail_segments.geojson
  • named_trails.csv
  • trailheads.geojson
  • areas.geojson
  • stewards.csv
